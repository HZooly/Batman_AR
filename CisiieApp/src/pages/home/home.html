Wikitude / IONIC2 Starter App

Please remember to: Add the Wikitude Plugin to this project with: ionic plugin add https://github.com/Wikitude/wikitude-cordova-plugin.git Obtain a (free) licence key and add it to src/app/app.component.ts

For detailed instructions, please look at the instructions on https://market.ionic.io/starters/wikitude-ionic-2-starter-app

For Ionic development, take a look at the src/pages/ directory to add or change tabs, update any existing page or create new pages.

To learn more about how to develop your own AR worlds, please look at http://www.wikitude.com/developer/documentation/phonegap

Have fun!
