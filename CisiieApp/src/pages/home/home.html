Bienvenue sur mon application mobile.
Batman Mobile a été développé avec à l'aide de Ionic 2 et du plugin de Réalité Augmentée Wikitude .

Cette application a été conçue dans le cadre du cours de Développement Mobile encadré par Monsieur Samuel Cruz-Lara.

Vous pourrez retrouver les sources de ce projet sur GitHub.

Ionic est un framework basé sur Angular et Cordova permettant de générer des applications mobiles (iOS, Android, Windows Phone etc...) à partir de langages web.

J'ai préféré utiliser la version 2 de Ionic, afin d'intégrer plus facilement le plugin Wikitude, tout en écrivant du code en TypeScript. De plus, les performances en v2 sont améliorées.

Réalité Augmentée

Wikitude est un plugin Cordova permettant d'intégrer des éléments de réalité augmentée au sein d'une application Cordova.

J'ai utilisé un modèle 3D libre de droit en FBX, converti celui-ci à l'aide de Wikitude 3D Encoder en WT3.
